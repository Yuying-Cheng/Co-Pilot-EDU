# 探究式学习任务生成与交互质量评估系统

## 环境要求

- Python 3.8+
- 依赖包：`pip install PyQt5 anthropic python-pptx PyPDF2 matplotlib`

## 配置 API Key

```bash
export ANTHROPIC_API_KEY="sk-ant-..."
```

Windows:
```
set ANTHROPIC_API_KEY=sk-ant-...
```

## 启动

```bash
cd learning_system
python main.py
```

---

## 功能说明

### 📚 课件导入 & 任务生成
1. 点击"选择文件"上传 PPT / PDF / TXT 课件
2. 可选填写章节标题
3. 点击"提取知识点"→ 系统自动抽取核心知识点
4. 点击"生成任务"→ 自动生成4-5个探究式学习任务
5. 点击"保存任务"→ 本地保存

### 📝 任务管理 & 编辑
- 浏览已保存的所有章节和任务
- 支持修改任务标题、描述、交互要求、难度等
- 保存后立即生效

### 🤖 成果评阅
1. 填写学生学号和姓名
2. 粘贴对话记录（格式：每行以"学生："或"模型："开头）
3. 填写最终报告和反思
4. 点击"开始评阅"→ AI自动评分并生成评语

支持格式：
```
学生：分治法的基本思想是什么？
模型：分治法可以理解为...
学生：我理解分治法就是把大问题拆成小问题，这样对吗？
模型：...
```

也可以直接导入 `student_submission.json` 文件。

### 📊 学情统计
- 自动汇总所有已评阅的学生成绩
- 展示分数段分布、交互方式分布、各维度得分率
- 识别群体薄弱知识点
- 生成分析总结

---

## 数据存储

所有数据保存在 `data/` 目录下：
- `data/courses/` - 课程知识点和任务
- `data/submissions/` - 学生提交记录
- `data/scores/` - 评分结果

## 评分标准

| 维度 | 满分 | 说明 |
|------|------|------|
| 交互质量 | 50分 | ≥10轮、多种方式、有追问、有质疑 |
| 知识掌握 | 25分 | 核心知识点覆盖度和理解深度 |
| 成果呈现 | 15分 | 报告清晰度和逻辑性 |
| 学习反思 | 10分 | 真实的学习感悟和改进方向 |
